# Resume Website

A simple, modern, and clean resume website.

## Features

- Clean, minimalist design
- Responsive layout for all devices
- Easy to customize
- Links to LinkedIn and publications

## Customization

1. Edit `index.html` to update your personal information, experience, education, skills, and publications
2. Update the LinkedIn URL in the contact section
3. Add your publication links in the Publications section
4. Modify `styles.css` if you want to change colors, fonts, or spacing

## Viewing

Simply open `index.html` in your web browser, or serve it using any static file server.
